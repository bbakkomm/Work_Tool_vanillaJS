3. 업데이트 내역
  • v5.1 그룹 당, 디폴트 여부 및 중복 체크

    1. 그룹핑 키 값 입력

    2. 디폴트 키 값 입력

    3. 그룹핑, 디폴트 값 체크 기능 On / Off

    체크 On 일 때, 그룹 키 값과 디폴트 키 값이 DB 내 키 값과 비교하여 올바르지 않으면 조건 문구 노출

  • 그룹 키, 디폴트 키 DB와 일치 시킨다.

  • 그룹핑과 디폴트 조건 로직

    1. 1개의 그룹 당, 디폴트 1개가 무조건 존재해야 합니다.

    2. 디폴트 0개 이거나 2개 이상일 때, 조건 문구 노출

    ** 디폴트는 [ -, O ]로 구성되어 있어야 정상 동작 가능
